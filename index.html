pushpot

simply implemented single-unique repository with node.js

no database. no auth.

supports CORS & JSONP

Install & Run

  1. git clone
  2. npm install
  3. node app.js -or- forever start -w app.js [recommended]

Usage

  1. create repo (returns uuid): http://yourDomain/?data=json string
  2. read repo : http://yourDomain/get/uuid
  3. read repo with JSONP : http://yourDomain/get/uuid?callback=your function name
  4. update repo : http://yourDomain/uuid/?data=json string
  5. update repo with JSONP : http://yourDomain/uuid/?callback=your function name&data=json string
